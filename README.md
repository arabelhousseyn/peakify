
## Peakify

peakify is a CRM solution

### requirments : 
 - php 8
 - mongodb global installation and mongodb extension installation in the php environment [https://www.php.net/manual/en/set.mongodb.php]

### run project :

 - composer install
 - in the command line  : cp .env.example .env
 - npm install
 - npm run dev
